Detailed Description

This visitor exports the Entity object structure into a GraphViz input file. A graphic representation of the object structure can then be created using the visitor.

See also
https://www.graphviz.org for DOT file notation and tools to generate graphics.

The manipulate method is inherited from EntityManipulator and since it is pure virtual (abstract) method there, it has to be overridden here. Otherwise, this class would also be abstract and you couldn't instantiate it.

Here, the GraphVizMarshaller just writes the propertes of the Entity into the output stream in GraphViz DOT format. If the Entity has children, then all those are also visited by this marshaller.
