Detailed Description

The interface class EntityManipulator defines what EntityManipulators do. They manipulate entities. There's no cpp file for this class, since the manipulate method is pure virtual and does not need an implementation. Interface classes usually just contain public pure virtual methods, no data. This class and the subclasses of it implement the Visitor design pattern together with the Entity class. Manipulators visits Entity object structures.

See also
https://en.wikipedia.org/wiki/Visitor_pattern
