Detailed Description

The XMLMarshaller class implements the EntityManipulator interface. Thus it is-a-kind-of EntityManipulator. It overrides the manipulate method, a pure virtual in EntityManipulator. What is marshalling: https://en.wikipedia.org/wiki/Marshalling_(computer_science)

void XMLMarshaller::manipulate ( EntityComposite & entity,
int level )
virtual

The manipulate method is inherited from EntityManipulator and since it is pure virtual (abstract) method there, it has to be overridden here. Otherwise, this class would also be abstract and you couldn't instantiate it.

Here, the XMLMarshaller just writes the propertes of the EntityComposite into the output stream in XML format. Then the marshaller is sent to visit the children of this Entity.
